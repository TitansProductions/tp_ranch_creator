# What has been updated?

## Changelog - (Added, Fixed, Removed):

1. Fixed an issue with objects and animals when placing and tried to relocate them, the animals and objects would be having a collision enabled and causing bugs and flickering.
2. Fixed chicken animal placement issue, the chicken because they are very small, they require `PlaceEntityOnGroundProperly` native function, otherwise their placement will be broken.
3. Fixed some data that returned invalid such as `AddMemberRequiredJobs` when set false, the option would return as `AddMemberRequiredJobs = { false }` than `AddMemberRequiredJobs = false` and feedbag, corn delivery and feeding positions data would return with an `[1]` parameter included which should not.
4. Fixed herding positions, they would set Z position to -1.0 which the position for walking was never displayed because it went underground.
5. Fixed herding wolf spawn positions from 1.0 to 0.0, same for animal spawning positions because in TP Ranch it could cause issues.
6. Improved the display for many data because they should be displayed as `{x,y,z,h,pitch,roll,yaw} than { x + new underline + y + new underline + z ...}
7. Fixed bucket for milking issue, it was extremely difficult to place the object under the cow or goat, by fixing it, i had to disable the collision of the spawned entity
