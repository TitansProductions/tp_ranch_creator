# What has been updated?

## Changelog - (Added, Fixed, Removed):

1. Fixed an issue on selected animals switches, even when the NUI was closed and reopened as new ranch creation, it would still display the same animals as selected from the previous ranch that was in progress when all of them should not be selected.
2. Added the possibility to edit existing ranches directly from `tp_ranch` script by using /openrancheditor [ranch id] - in case you want to create a new ranch, you don't add any ranch id.
3. Addeed an extra trigger event that is required to be used on `tp_ranch` when the player has a ranch editor active to hide all animals and objects from any ranches so this person can edit an existing ranch properly, otherwise ranch and ranch editor animals and objects will conflict with each other.
